# 导入导出接口约定

> 对应任务：`CHARTDB-P7-002`
> 适用范围：SQL、DBML、Smart Query metadata、backup 和 deterministic SQL export。

## 1. 总原则

导入导出流程必须保持本地优先：默认不连接生产数据库，不上传 schema，不要求账号或数据库密码。用户输入的 SQL、DBML、metadata 和 backup 文件只在浏览器本地解析，除非用户显式启用 AI-assisted export 或自托管 Gateway。

## 2. ImportResult

所有 dialect importer 应返回同构结果：

```ts
type ImportResult = {
    diagram: Diagram;
    confidence: 'high' | 'medium' | 'low';
    diagnostics: Diagnostic[];
    warnings: DialectWarning[];
    unsupportedObjects: UnsupportedObject[];
    sourceMap?: DialectSourceMap;
    sourceDialect: string;
};
```

要求：

- `warnings` 必须描述降级、fallback、语义不完整或需要人工审查的对象。
- `unsupportedObjects` 必须记录对象类型、名称、原因和是否被忽略。
- `confidence` 表示 importer 对本次解析完整性的判断；旧 wrapper 或未显式声明时默认 `medium`。
- `diagnostics` 记录面向用户或后续调试的结构化诊断，至少包含 `code`、`severity` 和 `message`。
- `sourceMap` 应尽量把 table、field、relationship、index 和 custom type 映射回源 statement 或源行号。
- importer 不得因为部分对象不支持而静默成功。

## 3. Import Preview

导入 Preview 是写入 IndexedDB 前的固定步骤。

Preview 至少展示：

- tables、relationships、customTypes、warnings 和 unsupportedObjects 数量。
- confidence 等级和 diagnostics 提示。
- 目标方言和使用的 fallback。
- parser error 或 `IMPORT_PREVIEW_PARSE_FAILED` 等结构化错误。

用户确认前不得创建、替换或删除 diagram 数据。

## 4. ExportResult

所有 deterministic exporter 应返回同构结果：

```ts
type ExportResult = {
    output: string;
    warnings: DialectWarning[];
    unsupportedObjects: UnsupportedObject[];
    riskLevel: 'low' | 'medium' | 'high';
    targetDialect: string;
};
```

要求：

- 出现类型降级、schema 到 database 语义转换、custom type 丢失或 unsupported syntax 时，`riskLevel` 不得低于 `medium`。
- 已知会生成错误 SQL 的情况，`riskLevel` 不得低于 `high`。
- AI-assisted export 不能替代 deterministic exporter 的风险标记。

## 5. DBML 与 SQL 方言

DBML wrapper 与 SQL dialect wrapper 共享 `src/dialects/common` contract。各方言能力以 `docs/方言能力矩阵.md` 和 `src/dialects/*/capabilities.ts` 为准。

当 MariaDB、CockroachDB 或其它相邻方言使用 fallback 时，结果必须在 warning 中明确标注 fallback，不得展示为完整原生支持。

## 6. Backup 导入导出

Backup 文件使用 `chartdb.backup` 格式，格式细节见 `docs/备份恢复格式.md`。

恢复前必须校验：

- `format`、`schemaVersion`、`createdAt`、`source` 和 `diagramCount`。
- `diagramCount` 与 `diagrams.length` 一致。
- diagram 基础字段完整。

校验失败时必须给出可理解错误，不写入 IndexedDB。

## 7. AI-assisted Export

AI-assisted export 默认关闭。启用时必须遵守 `docs/安全模型与AI边界.md`：

- Disabled：不能发起模型请求。
- BYOK Session：密钥只保存在运行时内存，刷新后失效。
- Self-hosted Gateway：浏览器只保存非敏感 endpoint 和 model name。

发起请求前必须向用户说明将发送的 schema 摘要范围。
